# `OrderedList` component
Astronaut:
```javascript
OrderedList() (
    ListItem() (),
    ListItem() (),
    ListItem() ()
)
```

HTML:
```html
<ol>
    <li></li>
    <li></li>
    <li></li>
</ol>
```

A list consisting of items that are represented in a particular order. The list should contain [`ListItem`](listitem.md)s.
