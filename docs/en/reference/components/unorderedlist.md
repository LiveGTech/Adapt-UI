# `UnorderedList` component
Astronaut:
```javascript
UnorderedList() (
    ListItem() (),
    ListItem() (),
    ListItem() ()
)
```

HTML:
```html
<ul>
    <li></li>
    <li></li>
    <li></li>
</ul>
```

A list consisting of items that are represented in no particular order (commonly known as _bullet points_). The list should contain [`ListItem`s](reference/components/listitem.md).
